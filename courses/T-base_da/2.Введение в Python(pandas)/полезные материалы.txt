Работа с jupyter 
Установите Anaconda для работы
Полезные утилиты
Нужно запустить из терминала эти команды:
pip install jupyter_contrib_nbextensions
pip install jupyter_nbextensions_configurator
pip install yapf
jupyter contrib nbextension install --user
jupyter nbextensions_configurator enable --user

Затем запустить вот эти команды:

jupyter nbextension enable notify/notify
jupyter nbextension enable toc2/main
jupyter nbextension enable execute_time/ExecuteTime
jupyter nbextension enable freeze/main
jupyter nbextension enable hide_header/main
jupyter nbextension enable codefolding/main
jupyter nbextension enable code_prettify/code_prettify

Полезные клавиши
Если ячейка выделена зеленым цветом, то вы находитесь в ячейке, если синим — то в просто с самом файле.

Переход из зеленого в синий — клавиша esc, из синего в зеленый — клавиша enter

Запуск ячеек осуществляется с помощью shift+enter (=запустить текущую и перейти в следующую) или ctrl+enter (=запустить текущую)

В «синем» режиме доступны горячие клавиши, основные из них:
a - добавить ячейку выше
b - добавить ячейку ниже
x - вырезать ячейку
v - вставить ячейку

Зажимая shift, можно выделить несколько ячеек разом

Shift + m - слить ячейки

Можно поменять тип ячейки (m - markdown, y - код)

Работа с текстом (видеоинструкция)
Полезно уметь быстро работать с текстом

ctrl + / - закомментировать строку/выделение
ctrl - перескакивать слова. Можно использовать с shift, чтобы выделять
Зажатый alt активирует прямоугольный мультикурсор
Зажатый ctrl активирует мультикурсор
